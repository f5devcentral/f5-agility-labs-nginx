Class 17 - NGINX Ingress Controller
====================================

This class provides a comprehensive walkthrough of leveraging NGINX Ingress Controller within a Kubernetes environment.
It showcases a variety of use cases, demonstrating how to effectively implement and manage gateway services for microservices architectures.

The lab includes step-by-step instructions, configuration examples, and practical scenarios that illustrate the powerful features 
of NGINX Ingress Controller, such as traffic management, security enhancements, and load balancing.

Expected time to complete: **2 hours**

This lab covers the following topics:

- Deploy NGINX Ingress Controller
- Basic Ingress Controller with URI-based routing and TLS offload
- Advanced Layer 7 routing
- JWT Token authentication
- Traffic splitting
- Access control
- Rate limiting
- NGINX App Protect WAF
