Class 18 - Mastering Cloud-Native App Delivery - ADCaaS (NGINXaaS)
==================================================================

Take your cloud infrastructure to the next level in this 2-hour hands-on lab featuring F5 NGINXaaS for Azure and Google Cloud! Dive into real-world use cases that push beyond the limitations of default cloud tools—master advanced traffic management, dynamic load balancing, and intelligent application routing capabilities that standard solutions can't deliver.__.

Expected time to complete: **2 hours**

This lab covers the following topics:

- Basic Ingress Controller with URI-based routing and TLS offload
- Advanced Layer 7 routing
- JWT Token authentication
- Traffic splitting
- Access control
- Rate limiting
